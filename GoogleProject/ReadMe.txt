The command to execute from command prompt
===========================================
A batch file (RunMe.bat) has been created in project level. Just double click on RunMe.bat file.

OR

Go to command prompt and give the below command.
[*DIR]\packages\NUnit.ConsoleRunner.3.8.0\tools\nunit3-console.exe [*DIR]\GoogleProject\bin\Debug\AssignmentProject.Tests.dll --result=TestResult.xml

Here::: *DIR= C:\Users\user\source\repos\GoogleProject

Test Report
===========
The Test Report will be available in location we have provided.
In above command TestResult.xml file will be created in specified location.
